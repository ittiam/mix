<h3 align="center">MIXUP</h3>
<p align="center">
  更易上手的前端构建工具
</p>

## 特性

- 简化 webpack 的配置，更人性化的配置参数
- 快速搭建项目且无需重复安装依赖，基于 webpack 4
- 生成的配置完全兼容 webpack 的命令行工具


### 使用 mixup 命令行工具

```shell
npx create-mixup-app my-app
cd my-app
npm start
```
